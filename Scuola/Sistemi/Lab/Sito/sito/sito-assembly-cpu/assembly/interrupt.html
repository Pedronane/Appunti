L'interrupt DOS o INT 21

L'interrupt 21h e uno degli interrupt piu usati del linguaggio assembly per la gestione delle operazioni di sistema. Questo interrupt fornisce una serie di funzionalita utili, come l'input e l'output, la gestione della memoria e delle stringhe, e la terminazione del programma.

Come funziona

Quando si vuole utilizzare una funzione specifica dell'interrupt 21h, si carica un valore (chiamato funzione) nel registro AH: ogni funzione ha un codice specifico che determina l'azione da eseguire ed in seguito si chiama l'interrupt 21h che permettera l'esecuzione della funzione.

Esempi dell'interrupt

01h: Legge un carattere dallo standard input 

        
          MOV AH, 01h  ; Carica la funzione 01h
          INT 21h      ; Chiama l'interrupt 21h
          ; Il carattere letto sar√† nel registro AL

02h: Scrive un carattere sullo standard output 

        
          MOV AH, 02h   ; carica la funzione 02h
          MOV DL, 'A'   ; Carattere da scrivere
          INT 21h       ; Chiama l'interrupt 21h

09h: Stampa una stringa 

        
          MOV AH, 09h       ; Carica la funzione 09h
          MOV DX, OFFSET msg ; Carica l'indirizzo della stringa in DX
          INT 21h           ; Chiama l'interrupt 21h

4Ch: Termina il programma 

        
          MOV AH, 4Ch   ; Carica la funzione 4Ch
          MOV AL, 00    ; Codice di ritorno
          INT 21h       ; Chiama l'interrupt 21h
