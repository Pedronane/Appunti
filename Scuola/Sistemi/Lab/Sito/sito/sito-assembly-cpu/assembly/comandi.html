Esempi di Comandi Assembly

MOV: Trasferisce dati tra registri, memoria e registri, o registri e memoria. 

        
          MOV AX, BX  ; Trasferisce il contenuto di BX in AX
          MOV [1b44h], AX ; Memorizza il valore di AX nell'indirizzo di memoria 1b44h

ADD: Somma valori nei registri o tra registri e memoria. 

        
          ADD AX, BX  ; Somma il valore di BX ad AX
          ADD [1b44h], AX  ; Somma il valore di AX al valore nell'indirizzo di memoria 1b44h

SUB: Sottrae il valore di un registro o di un'area di memoria da un altro registro. 

        
          SUB AX, BX  ; Sottrae il valore di BX da AX

CMP: Confronta due valori senza modificarli. Aggiorna i flag per riflettere il risultato. 

        
          CMP AX, BX  ; Confronta AX con BX

JMP: Esegue un salto incondizionato a una specifica etichetta nel codice. 

        
          JMP ETICHETTA  ; Salta all'etichetta ETICHETTA

CALL: Chiama una subroutine, memorizzando l'indirizzo di ritorno nello stack. 

        
          CALL STAMPA  ; Chiama la subroutine STAMPA

RET: Ritorna da una subroutine, utilizzando l'indirizzo di ritorno memorizzato nello stack. 

        
          RET  ; Ritorna dalla subroutine
